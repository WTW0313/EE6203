学习建议：

Lecture Notes上的概念和例题过一遍，避免概念混淆，然后通过习题来巩固解题方法。这门课的考试题型比较固定，一般不太会有陌生的题，复习的时候可以直接参考往年考题。考试前多做题，保持良好的做题手感很重要，是相对来说复习起来比较轻松的一门课。五门课选手可以选择配合Process Control来一起学习，对于没有控制理论基础的同学相对友好一些，学了这门课能够帮助理解PC的课件。
